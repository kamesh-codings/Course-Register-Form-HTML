Welcome to the repository for the **Course Registration Form** project. This project is a practical application of front-end development, focusing on creating a clean, user-friendly, and functional web form for data collection.

---

## 📖 About
This project demonstrates the core principles of web form design and user input handling. It serves as a study in gathering structured data efficiently while maintaining a professional aesthetic. Key focuses include:

*   **Form Structure:** Implementation of varied input types (text, email, selection, radio buttons) to ensure comprehensive data capture.
*   **Accessibility & Usability:** Using proper labels and field grouping to create an intuitive experience for the user.
*   **CSS Styling:** Applying modern layout techniques to make the form visually appealing and easy to navigate.
